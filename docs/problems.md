# Honokaが解決する課題

## 概要

Honokaは「読書を習慣化したいけどできない」人のためのアプリです。

---

## 課題1: 未読が多すぎて何を読んでいないか把握できない

### 具体的な困りごと
- 本は買うけど、どれが未読かわからなくなる
- 「これ読んだっけ？」と本屋で悩む
- 次に何を読むか決められない

### Honokaの解決策
| 機能 | 説明 |
|------|------|
| **書斎** | 未読・読了・中断を分けて管理 |
| **本屋** | 気になる本をウィッシュリストとして登録 |
| **カバン** | 今読んでいる本だけを表示 |

→ 本の状態が一目でわかり「何を読んでないか」が明確になる

---

## 課題2: ついスマホを触ってしまい読書時間が確保できない

### 具体的な困りごと
- 読もうと思っても SNS やゲームに流れる
- 読書を始めるきっかけがない
- 継続できない

### Honokaの解決策
| 機能 | 説明 |
|------|------|
| **読書タイマー** | スマホで読書時間を計測（スマホが読書の相棒に） |
| **ストリーク** | 連続読書日数を可視化し、途切れさせたくない気持ちを活用 |
| **読書カレンダー** | GitHub草のように読書履歴を可視化 |
| **セレブレーション** | 読了時のアニメーションで達成感を演出 |

→ スマホを「読書の敵」から「読書の味方」に変える

---

## ターゲットユーザー

- 本は好きだけど読む時間が取れない人
- 未読が溜まっている人
- 読書習慣をつけたい人
- スマホ時間を読書時間に変えたい人

---

## Honokaの価値

> 「スマホで本を管理し、スマホで読書時間を計る」

スマホを触る習慣を逆手に取り、**スマホを開くたびに読書を思い出す**仕組みを提供する。
